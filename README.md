# ChildProofLock – Keyboard + Mouse Lock (AutoHotkey v2)

This tool locks your Windows keyboard and mouse so a child cannot interact with the system.  
Unlocks only using **Ctrl + Shift + U**.

### Features
- Full keyboard lock (except unlock combo)
- Full mouse lock (left, right, wheel, extra buttons)
- Optional mouse movement freeze
- Unlock instantly with **Ctrl+Shift+U**
- Safe, lightweight AutoHotkey v2 script

###  How to Run
Install AutoHotkey v2:
https://www.autohotkey.com/

Run:
Press:
- **F12** → Lock inputs  
- **Ctrl + Shift + U** → Unlock

### Convert to EXE
Install AutoHotkey v2, then right-click the script →  
**Compile → Convert to EXE**
